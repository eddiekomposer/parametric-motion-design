# Parametric Motion Design

独立的图片傅立叶动效工具，不依赖原来的 Motion Design Tool。

## 功能

- 上传任意图片
- 按 `biran0079/circle` 的思路提取等值线点云
- 点云采样后构造 MST 路径
- 路径平滑与 Ramer-Douglas-Peucker 简化
- 使用 `FFT(path_x + i path_y)` 生成傅立叶公式
- 复用 Math Curve Loaders 的粒子、尾迹、循环、呼吸、旋转、轨迹粗细参数生成动效
- 新增“正弦波”动效：普通 stroke 正弦线默认水平叠放，支持 taper 收尾和线条随机性，默认线色 `#8F959E`
- 新增“示波器”动效：用多个李萨如图形生成相位驱动动画，支持转换/并列模式、图形参数、旋转、颜色、错峰刷新和鼠标滑过力反馈
- 新增“细胞”动效：根据上传参考图的前景 mask，用圆形填满轮廓内部；中心圆形更大、边缘圆形更小，支持换位、闪烁、生长和旋转四种动效
- 新增“变形”动效：支持上传多张关键帧图片，按每张图的前景连通域划分独立形状，并让形状轮廓之间用自然 morph 曲线和呼吸感过渡

## 使用

直接打开 `index.html`，或在本目录启动任意静态服务器。

```sh
python3 -m http.server 8765
```

然后访问 `http://127.0.0.1:8765/`。
